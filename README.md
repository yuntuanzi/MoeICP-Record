
# 🎀 萌ICP备现存备案记录 (MoeICP-Record)

MoeICP-Record 爬取了2023年至2026年2月13日期间的现存备案记录。核心实现备案信息的**存储、查询与可视化展示**。

## 📄 项目基础信息
技术栈：PHP 7.4+、SQLite 3+

## ✨ 项目特点
1. **轻量易部署⚡**
   - 基于PHP+SQLite开发，无外部依赖，无需配置复杂的数据库服务
   - SQLite为文件型数据库，开箱即用，单文件即可完成部署
   - 占用服务器资源少，支持低配服务器/虚拟主机运行

2. **多格式数据存储💾**
   - 同时提供SQLite数据库和JSON文件双存储方案
   - 数据库满足高效查询需求，JSON文件便于跨平台读取和数据交换
   - 支持数据格式互转，兼顾性能与易用性
