## 🎀 项目介绍

**MoeICP-Record（萌ICP备案记录）** 是一款轻量级的备案记录管理系统，专门爬取并整理了**2023年至2026年2月13日**期间的现存备案记录。
系统核心聚焦于备案信息的**存储、精准查询与直观可视化展示**，为备案数据管理提供极简且高效的解决方案！

## 📄 项目基础信息

| 核心组件 | 版本要求 | 备注 |
|----------|----------|------|
| PHP | ≥ 7.4 | 核心运行环境 |
| SQLite | ≥ 3 | 轻量数据库引擎 |
| 数据格式 | SQLite / JSON | 双格式支持，兼顾易用性与扩展性 |

> 💡 网站主体采用SQLite实现开箱即用，预留JSON格式专为二次开发提供便利
