## 🎀 项目介绍

**MoeICP-Record（萌ICP备案记录）** 是一款轻量级的备案记录管理系统，爬取并整理了**2023年至2026年2月13日**的现存备案记录。

## 📄 项目基础信息

| 核心组件 | 版本建议 |
|----------|----------|
| PHP | ≥ 7.4 |
| SQLite | ≥ 3 |
| 数据格式 | SQLite / JSON |

> 网站采用SQLite实现开箱即用，预留JSON格式方便二次开发

### 🎯 核心功能
- **轻量易部署** - 基于PHP+SQLite开发，无需配置复杂数据库服务
- **多格式存储** - 同时支持SQLite数据库和JSON文件双存储方案
- **精准查询** - 检索2023-2026年的备案记录

### 📋 部署要求
- PHP 7.4 及以上版本

### 🔧 部署步骤
1. 下载源码并解压至网站根目录
2. 直接访问域名/IP即可
3. 无需额外配置
